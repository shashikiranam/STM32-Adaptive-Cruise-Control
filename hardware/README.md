# Hardware

This directory contains the hardware documentation for the STM32-Based Adaptive Cruise Control and Collision Avoidance System.

## Hardware Modules

- STM32 NUCLEO-G071RB Development Board
- HC-SR04 Ultrasonic Sensor
- L298N Motor Driver
- DC Motor
- Power Supply
- Jumper Wires
- Breadboard

## Documentation

- COMPONENTS.md – List of hardware components
- PIN_CONNECTIONS.md – Complete wiring and pin mapping
- POWER_REQUIREMENTS.md – Power specifications

The hardware architecture is designed for modularity, allowing easy integration with additional sensors and automotive peripherals.
