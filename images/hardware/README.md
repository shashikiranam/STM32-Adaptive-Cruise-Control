# Hardware Images

This directory contains hardware diagrams and board connection illustrations for the STM32 Adaptive Cruise Control and Collision Avoidance System.

# Hardware Setup

The hardware setup includes all components required to implement the Adaptive Cruise Control prototype.

Components:

- STM32 NUCLEO-G071RB
- HC-SR04 Ultrasonic Sensor
- L298N Motor Driver
- DC Motor
- 16×2 LCD
- LEDs
- Power Supply

---

# Wiring Diagram

The wiring diagram illustrates the electrical interconnections between the STM32 microcontroller and external peripherals.

Connections include:

- Ultrasonic Sensor
- Motor Driver
- LCD
- LEDs
- Power Supply

# Board Connections

The board connection diagram provides the STM32 GPIO assignments used throughout the project.

Interfaces include:

- GPIO
- Timer
- PWM
- UART
- LCD Interface

# Purpose

These hardware illustrations help understand the physical implementation of the Adaptive Cruise Control system and simplify hardware replication.
