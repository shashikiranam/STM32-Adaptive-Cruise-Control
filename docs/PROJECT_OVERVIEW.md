
# Project Overview

## Project Title

STM32-Based Adaptive Cruise Control and Collision Avoidance System

## Objective

The objective of this project is to develop an Adaptive Cruise Control (ACC) system capable of automatically adjusting the vehicle speed based on the distance to the object ahead.

The system continuously monitors the surrounding environment using an ultrasonic sensor and controls the vehicle speed using PWM-based DC motor control.

## Key Features

- Adaptive Cruise Control
- Collision Avoidance
- Distance Measurement
- PWM Motor Speed Control
- Real-Time Monitoring
- Embedded C Firmware
- STM32 Microcontroller

## Applications

- Autonomous Vehicles
- Driver Assistance Systems (ADAS)
- Robotics
- Smart Mobility
- Automotive Electronics
