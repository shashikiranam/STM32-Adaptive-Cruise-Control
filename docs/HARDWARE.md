# Hardware

## Components Used

| Component | Purpose |
|-----------|---------|
| STM32 Blue Pill | Main Controller |
| HC-SR04 Ultrasonic Sensor | Distance Measurement |
| L298N Motor Driver | DC Motor Control |
| DC Motor | Vehicle Motion Simulation |
| Power Supply | System Power |

## Hardware Functions

### STM32

Processes sensor data and controls motor speed.

### HC-SR04

Measures the distance between the vehicle and an obstacle.

### L298N

Controls the DC motor using PWM signals.

### DC Motor

Represents the vehicle speed.
